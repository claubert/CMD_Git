git ignore - > É onde você não irá subir para o github e deverá criar na raiz do projeto um .gitignore. Dentro do .gitignore escreva o que não quer ser versionado

Criando um arquivo e subir para o git -> Primeiro precisa criar um arquivo e dentro do aquivo clicar com o botão direto e escolher a opção "open git bash here". Irá abrir uma janela e tem que digitar o comand: git init para iniciar. 
- Depois digitar git status para ver os aquivos dessa pasta
- Adicionar os arquivos um a um -> git add ".e o nome do arquivo"
- Adicionar todos os arquivos - > git add .
- Subir a versão do projeto - git commit -m" colocar o mentário"
- Conectando o projeto ao github - > Deve crair o arquivo dentro do githube e no git digitar o comando - git remote add origin "Aqui dentro pegar o link do github do projeto que foi criado"
- Agora deve digitar o comando git push --set-upstream origin master
 - Agora para enviar o comando é git push --set
